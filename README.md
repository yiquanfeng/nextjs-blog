## Spriple's Blog by Nextjs
you can also use it to your blog
first install the dependencies
```shell
npm install
```
then you can move you markdwon file to the posts/ directory to let the ts script to find it
finally you can use this to preview it on your local network
```
npm run dev
```
then view it on http://localhost:3000
if you want deploy it to your github page
you can just make a github repo in https://www.github.com
then push the whole project to the remote repo
then my action file can build and deploy it
